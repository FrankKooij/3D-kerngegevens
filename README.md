# 3D-kerngegevens

De werkgroep 3D-kerngegevens werkt aan de representatie van de leefomgeving in een stelsel van 3D-basisgegevens. Hieronder kunnen vallen:
- Bebouwde bovengrond
- Ondiepe ondergrond
- Diepe ondergrond
- Niet-fysieke objecten (zoals bestemmingsplannen, kadaster, etc.)
